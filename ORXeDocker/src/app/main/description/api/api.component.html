Steps to use this Component Step 1 : Install Dependencies
Node (>6.0.0) Npm (>3.8.6) Step 2 : Installation
# Install compodoc with npm
$ npm install -g @compodoc/compodoc Step 3 : # Run compodoc in your project (generated with Angular CLI for example) and serve it
$ compodoc -p src/tsconfig.app.json -s
